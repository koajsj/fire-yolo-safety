# 答辩演示指南

## 演示前准备

1. 确认后端和前端都已启动。
2. 打开 `http://127.0.0.1:5173`。
3. 将演示图片和短视频放入 `demo_files/`，便于现场选择。
4. 如果有自训练模型，将 `best.pt` 放入 `weights/` 并重启后端。
5. 建议提前完成一次图片或视频检测，让首页图表有数据。

## 推荐演示流程

### 1. 首页仪表盘

展示安全指数、今日检测次数、异常检测次数、风险等级分布、类别分布、每日趋势、检测方式占比和当前模型状态。

重点说明：系统不是单纯目标检测，而是结合风险等级、统计分析和预警提示的消防安全风险分析系统。

### 2. 图片检测

进入“图片检测”，上传 `jpg/jpeg/png` 图片，点击开始检测。

展示内容：

- 原图和结果图
- 风险等级
- 检测目标总数
- 火焰、烟雾、抽烟、人员数量
- 最高置信度类别和置信度
- 检测耗时
- 中文风险提示

### 3. 视频检测

进入“视频检测”，上传较短视频，点击开始检测。

说明视频检测会逐帧处理，耗时比图片检测更长。检测完成后展示结果视频、整体风险等级、目标统计和风险摘要。

### 4. 摄像头检测

进入“摄像头检测”，点击打开摄像头并允许浏览器权限。

展示内容：

- 当前帧检测结果
- 当前风险等级
- 主要类别
- 风险提示文本
- 高风险或中风险预警
- 持续风险状态和恢复低风险提示

演示结束后点击关闭摄像头，确认页面不再继续检测。

### 5. 检测记录

进入“检测记录”，展示新增记录。

可演示搜索、筛选、查看详情和删除记录。详情中重点展示风险等级、检测耗时、类别统计和结果文件路径。

### 6. 系统设置

进入“系统设置”，展示置信度阈值。

说明：阈值越高，检测越严格；阈值越低，检测越敏感。保存后会影响后端检测过滤。

### 7. 模型状态

进入“模型状态”，展示当前模型文件、来源、运行设备、阈值、加载状态和类别信息。

如果当前使用 `yolov8n.pt`，需说明它仅用于流程演示，不代表消防专用识别能力。如果使用 `best.pt`，说明这是自训练消防安全模型。

## 现场建议

- 视频尽量使用 30 秒以内的短视频，避免等待过久。
- 摄像头演示前先确认浏览器权限和设备占用情况。
- 如果没有消防专用模型，答辩时明确说明当前默认模型只用于展示系统流程。
- 先演示图片检测，再演示视频和摄像头，节奏更稳定。
